# R(0,1,0) — Complex Numbers ℝ(0,1,0) ≅ ℂ

One-generator Clifford algebra with `e1² = −1`. Isomorphic to the complex
numbers: scalar = real part, `e1` = imaginary unit `i`.

## Basis & metric

| 1   | e1  |
|-----|-----|
| +1  | -1  |

Signature: e1² = −1.

## Cayley table

|        | 1   | e1   |
|--------|-----|------|
| **1**  | 1   | e1   |
| **e1** | e1  | -1   |

## Geometry

| object        | formula                                            |
|---------------|----------------------------------------------------|
| real          | `a · 1`                                            |
| imaginary     | `b · e1`  (b · i)                                  |
| complex       | `a + b · e1`  — rendered as point `(a, b)`         |
| modulus       | `|z| = √(a² + b²)`                                 |
| conjugate     | `~z = a − b · e1`                                  |
| Euler rotor   | `exp(θ·e1) = cos θ + sin θ · e1`                   |
| rotation      | `z′ = R · z`  — left-multiplication                |

## Sub-algebras

| sub-algebra                  | blades   |
|------------------------------|----------|
| Scalars (real line ℝ)        | `1`      |
| Full algebra ≅ ℂ             | `1, e1`  |

## Notes

- Every value classifies as a finite point in the complex plane — scalars sit
  on the real axis, `e1` multiples on the imaginary axis.
- Reverse maps to complex conjugate (the GA reverse leaves grade-1 alone for
  this signature, so the adapter uses `Conjugate` instead).
