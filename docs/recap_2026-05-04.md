# Project Recap — 2026-05-04

## GA Constructor

A web-based 2D geometric construction tool built on Projective Geometric Algebra (PGA(3,0,1)).

## Architecture

Expression-driven graph — the user types expressions in a panel on the left; each expression becomes a node in a dependency graph that is evaluated and rendered on a canvas.

## Key files

| File | Role |
|---|---|
| `src/pga.js` | PGA primitives: `point2D`, `idealPoint`, `toEuclidean`, `lineBaseAndDir` using ganja.js |
| `src/graph/parseExpression.js` | Parses text expressions into typed node definitions |
| `src/graph/evalExpr.js` | Evaluates math expressions (supports scalar refs, sin/cos/PI…) |
| `src/graph/nodeTypes.js` | Defines each node type's `compute()` function |
| `src/graph/evaluate.js` | Topological evaluation of the whole graph |
| `src/useGraph.js` | Main state hook — items list, animation loop, derived maps |
| `src/ExpressionPanel.jsx` | Left sidebar UI — inputs, color swatches, interval/position sub-rows |
| `src/Canvas.jsx` | Canvas rendering + pan/zoom/drag interactions |

## Supported expression types

| Syntax | Type | Description |
|---|---|---|
| `A = 3.5` | scalar | Number, animatable with an interval |
| `A = point(x, y)` | freePoint | Draggable 2D point |
| `A = vector(vx, vy)` | vector | Free vector (draggable tail + tip) |
| `L = A & B` | joinLine | Line through two points (regressive product) |
| `M = L1 ^ L2` | meetPoint | Intersection of two lines (wedge) |
| `T = exp(G, s)` | motorExp | Motor from a line or vector |
| `R = T >>> P` | motorApply | Apply motor to a point/line (sandwich product) |

## UI features

- Color picker per expression
- Scalar animation: play/pause with configurable `(min, max, step)` interval
- Vector draw-position: static `(x, y)` or linked to a point node by ID
- Canvas: pan (drag), zoom (scroll), drag free points and vector tails/tips
